Scripting vector data in SVG.

This example demonstrate how to embed javascript into SVG. Switzerland has been divided into about 22'000 small catchment areas. Theses area of about 2 km2 have been organized in a hierarchical structure. So this is easy to get all upstream areas.

If you fly the mouse over a catchment area, you will highlight it's entire upstream area.

View the aare.svg (16 MB) source to see how this is done.

To download and have more information, go to "Einzugsgebietsgliederung Schweiz"
