Printing with GeoAdmin API.

If you want to print with GeoAdmin API you will have to set up a proxy relaying the request to and response from the print server.

As the printing process is not staless, you will have to manage the SRV cookie sent by the server so that your requests are directed to the right server.

View the print.js source to see how this is done.
