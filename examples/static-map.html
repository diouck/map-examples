Geoadmin Static Map Example

phantomjs, rasterize, heroku, static map

Using PhantomJs to rasterize map content.

Sometime you juste need to display a static map in a web page. With the help of a headless web browser like PhantomJS and a basic web service (here, Flask hosted on Heroku>, you may rasterize any web site.

Try pasting a GeoAdmin Permalink (normale or shortened) and generate the image (may take a few seconds).

Copy-paste this code in your web page
