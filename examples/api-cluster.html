GeoAdmin full OpenLayer build Example

vector, feature, stylemap, wfs, cluster, strategy, cleanup, geoadmin

Uses the GeoAdmin full OpenLayers build to use "exotic" features like the cluster strategy.

Not all classes of OpenLayers are inlcuded per default in the GeoAdmin API build, the cluster strategy for instance. If you want to use this class, you will have to use loader.js?mode=fulll to get a build with the full OpenLayers libraries (this build is about 25% bigger than the normal build).

This exampled has been adapted from OpenLayers

View the api-cluster.js source to see how this is done.

Hover over a cluster on the map to see the photos it includes.
