Heatmap GeoAdmin API Example

heatmap, Субботник, geoadmin, earthquake, wfs, wms

Recent earthquakes in Switerland as an heat map.

This example creates an heatmap from the latest earthquake in Switzerland. Obviously, Wallis is the most active region. Unfortunately, the WFS data do not hold the earthquake's magnitude.

The code for the heatmap is from fredj's openlayers-heatmap


The earthquake data are live from Schweizerische Erdbebendienst (both WMS and WFS).

View the heatmap.js source to see how this is done.
