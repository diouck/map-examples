WMTS Capabilities GeoAdmin API Example

wmts,capabilities, geoadmin, mf-chsdi,api, openlayers

Adding layers from WMTS GetCapabilities into GeoAdmin API.

This example show how to add WMTS layers from a GetCapabilities document. You just need to speicify that some configuration are missing with capabilitiesNeeded: true and point with url to a GetCapabilities document.

View the wmts-capabilities.js source to see how this is done.
