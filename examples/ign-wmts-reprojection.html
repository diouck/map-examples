projection, mercator, equirectangular, wmts, ign, openlayers

Mixing Mercator and Equirectangular layers in the IGN API.

Show how to overlay a Mercator WMTS layer ('Pixelkarte' and 'OSM' in this case) on top of the Map equirectangular projection in IGN API

Note, EPSG:90013 and EPSG:3857 are the same pseudo-mercator "projection", a number number is official, another is not! (guess which one, otherwise http://www.epsg-registry.org/).

View the ign-wmts-reprojection.js source to see how this is done.
