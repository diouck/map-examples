Combining swisstopo excellent maps with Google Map.

Many people find the Google Map API easier to use than the OpenLayers-based one provided by GeoAdmin.

This simple example show how to integrate layers from swisstopo. You have to get the tiles through French IGN API or use something like MapProxyto reproject the swisstopo tiles, as the projection used by these two APIs are not compatible. Note, you probably have to do this on-the-fly as caching tiles is forbbiden by swisstopo.

View the google.js source to see how this is done.
