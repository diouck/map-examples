GeoAdmin and Panoramio Example

vector, panoramio, photos, strategy, geoadmin, script, jsonp

Uses GeoAdmin to display photos from Panoramio.

Displaying geo tagged photos from Panoramio on GeoAdmin API. Using an OpenLayers.Strategy.BBOX, we get the 50 most popular photos fitting the map area each time the map is zoomed or paned.

View the panoramio.js and OpenLayers.Format.Panoramio.js source to see how this is done.
