Deep tree CatalogTree GeoAdmin API Example

catalogtree, tree, geoadmin, openlayers

More than 3 levels of hierachy in a GeoAdmin CatalogTree.

The CatalogTree is in no way limited to 3 levels. Arbitrary deep hierarchies may be displayed. This widget is really a tree, so it may also look like a real tree, and not like a super accordion like in map.geo.admin.ch

The custom tree is defined in the App.catalogConfig.geodata.

View the deeptree.js source to see how this is done.
