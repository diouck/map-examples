Porting Geoevents to the GeoAdmin API Example

geoevents,google map, cluster,geoadmin,api,control,geology,openlayers

From Google Map API to the GeoAdmin API.

This basic example was initially made with the Google Map API. It was ported to the GeoAdmin API to show it is not all too complicated to do this. The big benefit of using the GeoAdmin API is the possibility to display the various geology-related layers as background to the various geoevents.

Fly over the event location to display the details and click on it to jump to the webpage of the event.

View the geoevents.js source to see how this is done.
