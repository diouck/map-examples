Using GeoAdmin WMTS tiles with the popular Leaflet library (III)

After a basic demo using leaflet an geo.admin.ch tiles in (part I) and displaying GeoSJON data in (part II), we are now exploring Leaflet's draw plugin.

Some feature are already added to the map, you may choose to edit or delete them, or add your own. The features are then stored in HTML5 localStorage (reload the page to see). You may also want to reset evereything to the pristine state hitting the reset button.
