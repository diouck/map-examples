Loading Google Fusion Table into GeoAdmin Map.

Displaying data from Google Fusion Table service into the GeoAdmin. Fusion Table now offer the possibility to get a kind of JSON format instead of a CSV file which makes it easier to integrate a table into OpenLayers.

You can mouse over the feature to check it is real OpenLayers Features.

View the cross-domain.js source to see how this is done.
