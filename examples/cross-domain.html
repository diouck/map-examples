GeoAdmin API loading GeoJSON via JSONP

geoadmin,mf-chsdi,api,openlayers,vector,geojson,jsonp,cross-domain,xss,advanced

Loading GeoJSON data from a third-party website using the new OpenLayers.Protocol.Script.

Using the feature service from the GeoAdmin API returning serilized data, it is possible to display a vector layer on the map.
In this case, we use OpenLayers.Protocol.Script to make cross origin request.

View the cross-domain.js source to see how this is done.
