Fractional zoom GeoAdmin API Example

zoomlevel, unlimited zoom, scale, geoadmin, mf-chsdi,api, openlayers

Demonstrate the use of fractional (or non-discrete) zoom level with GeoAdmin API.

(zoom: )

Setting the map.fractionalZoom property to true allows zooming to an arbitrary level (between the min and max resolutions). This can be demonstrated by shift-dragging a box to zoom to an arbitrary extent (from the OpenLayers.org examples).

View the api-fractional-zoom.js source to see how this is done.
